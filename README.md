# Quick Sample Sort

Implementação do algoritmo QuickSampleSort em Java, baseada no artigo de Joseph Jaja (2000) *A Perspective on Quicksort*.  
O projeto simula a execução paralela com 4 processadores, comparando o melhor caso (subvetores iguais) e o caso médio (pivôs por amostragem).

# Referência
Jaja, J. (2000). A Perspective on Quicksort. Computing in Science and Engineering, 2(1), 43–49. IEEE.
